Margens e Padding.

Margens e Paddings no Bootstraping são aplicados de modo geral, baseado no eixo ou na direção.

Class Propriedades
m- Margem para todos os lados.
mx- Margem no eixo X (Esquerda e Direita).
my- Margem no eixo Y (Cima e Baixo).
mt- Margem Top (Cima).
me- Margem End, Right (Direita).
mb- Margem Bottom (Baixo).
ms- Margem Start, Left (Esquerda).
Class Propriedades
p- Padding para todos os lados.
px- Padding no eixo X (Esquerda e Direita).
py- Padding no eixo Y (Cima e Baixo).
pt- Padding Top (Cima).
pe- Padding End Right (Direita).
pb- Padding Bottom (Baixo).
ps- Padding StartLeft (Esquerda).
