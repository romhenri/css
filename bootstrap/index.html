Curso de Bootstrap

Bootstrap é um framework front-end que fornece estruturas de CSS para a criação de sites e aplicações responsivas de forma rápida e simples.

Estruturas com Contêineres.

Importante lembrar que Bootstrap é Mobile-First!

Isso Implica que inicialmente devemos pensar no layout mobile, e depois adaptar para experiência de desktop. As classes de containers são baseadas em width a partir das marcas. Exemplo: class="col-12 col-xxl-8 mx-auto". Neste trecho vemos primeiro a configuração do layout mobile primeiro (col-12), depois, a adaptação para telas a partir de 1320px (col-xxl-8).
